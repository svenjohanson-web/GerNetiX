# Runtimes

Runtimes beschreiben, in welcher technischen Umgebung ein modellbasiertes Lernprojekt ausgefuehrt wird.

Fuer das Tamagotchi gibt es zwei didaktische Runtime-Pfade:

- `browser_app/`: Browser App fuer Mac, PC und Mobile.
- `embedded/`: Embedded Board, zum Beispiel ESP32.

Mac, PC und Mobile sind beim Tamagotchi keine getrennten Lernpfade. Sie nutzen gemeinsam die Browser-App-Runtime.

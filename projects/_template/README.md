# Projektvorlage

Diese Vorlage zeigt, wo spaetere Projektanpassungen liegen.

Fuer ein echtes Projekt wird `_template` kopiert und umbenannt, zum Beispiel:

```text
projects/workshop-sensor-node/
```

Projektanpassungen duerfen nur deklarative Konfiguration und freigegebene Hooks enthalten. Die Basissoftware bleibt getrennt und wird nicht in Projektordner kopiert.

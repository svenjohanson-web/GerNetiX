# Dokumentationsthemen

Dieses Dokument sammelt die ersten Themenbereiche fuer die GerNetiX-Dokumentation.
Der Fokus liegt auf dauerhaftem Engineering-Wissen, nicht auf schneller Code-Erzeugung.

## 1. Zielbild und Prinzipien

- Engineering Knowledge Platform
- Leitsatz: Wissen ist teurer als Code
- Code implementiert Wissen
- Metamodell bewahrt Wissen
- Abgrenzung zu einer klassischen Webanwendung

## 2. Produkte

- Learning Platform
- Simple IDE
- Gemeinsame Wissensbasis
- Produktuebergreifende Traceability

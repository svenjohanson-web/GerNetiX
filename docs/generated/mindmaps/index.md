## Verwendung in VS Code

1. Eine Mindmap-Datei oeffnen.
2. Markdown Preview oder Mermaid Preview starten.
3. Die Mermaid-Mindmap ansehen.

Hinweis:

Mermaid zeigt die Mindmap an.
Die Beziehungen werden nicht automatisch aus YAML erkannt, sondern muessen spaeter aus YAML generiert werden.

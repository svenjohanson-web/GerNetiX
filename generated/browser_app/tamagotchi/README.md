# Generierter Browser-App-Code

Dieser Ordner ist fuer aus dem Tamagotchi-Modell erzeugten Browser-App-Code vorgesehen.

Generated-Code wird nicht manuell zur Quelle der Wahrheit. Aenderungen erfolgen am Modell unter `model/tamagotchi/`.

Die erste Version darf rein ereignisbasiert und ohne Persistenz sein. Spaetere Versionen koennen eine zeitgesteuerte State-Machine und Browser-Speicher nutzen.

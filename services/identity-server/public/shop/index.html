Software-Lizenz

GerNetiX Home Server Software

Lizenzgebundene Nutzung des GerNetiX Home Servers fuer eigene oder vorhandene Hardware. Dieses Produkt ist nur ein Platzhalter ohne Kaufabwicklung.

Einzellizenz

Home Server Lizenz - 12 Monate

Installation, Konfiguration, Updates, Cloud-Funktionen, KI und neue Integrationen.

Premium enthalten

Premium jaehrlich inkl. Home Server

Bei jaehrlicher Premium-Abrechnung ist die Home-Server-Nutzung als Nutzungsrecht enthalten.

Lizenz aktiv

  • Home Server installieren
  • neue Geraete hinzufuegen
  • Server konfigurieren
  • Updates erhalten
  • Cloud-Funktionen nutzen
  • KI nutzen
  • neue Integrationen aktivieren

Lizenz abgelaufen

  • bestehende Geraete funktionieren weiter
  • MQTT laeuft weiter
  • Home Assistant laeuft weiter
  • OTA fuer eingerichtete Modelle bleibt moeglich
  • keine neuen Geraete, Integrationen, Cloud-/KI-Funktionen oder Feature-Updates

Bestandsschutz: Eine abgelaufene Lizenz legt das lokale Zuhause nicht lahm. Sie stoppt nur Erweiterung, Cloud-Komfort und neue kostenpflichtige Funktionen.

    Provisioning

    Wie möchtest du das Board verbinden?

    Wähle USB, wenn du das Board noch nie mit GerNetiX verbunden hast, es per WLAN nicht gefunden wird oder bisher nur die minimale Inbetriebnahme-Provisionierung ausgeführt wurde. WLAN ist für bereits vollständig provisionierte und erreichbare Boards.

    Geführtes Provisioning

    Board suchen

    Wichtig: WLAN funktioniert nur mit bereits provisionierten Boards. Das Board muss die GerNetiX-Basissoftware bereits besitzen und im gleichen lokalen Netzwerk erreichbar sein. Für ein neues oder nicht erreichbares Board bitte USB wählen.

    Dieser Browser unterstützt Web Serial nicht. Verwende stattdessen das GerNetiX USB Helper Tool.

    Basissoftware

    Board per USB flashen

    GerNetiX schreibt jetzt die Basissoftware auf das erkannte Board. Erst nach einem erfolgreichen Flash kann das Board registriert und mit deinem Account verbunden werden.

    WLAN einrichten

    Board mit deinem WLAN verbinden

    Das Board sucht die WLANs selbst. SSID und Passwort werden weder an GerNetiX übertragen noch im Browser gespeichert – sie gehen ausschließlich per USB an das Board und werden dort lokal abgelegt.

    Wenn du diesen Weg nicht nutzen möchtest, kannst du das Captive Portal des Boards verwenden. Alternative in der Hilfe öffnen.

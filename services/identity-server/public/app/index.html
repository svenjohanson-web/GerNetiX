    Provisioning

    Board verbinden

    Per WLAN oder USB provisionieren

    WLAN übernimmt ein bereits provisioniertes Board aus deinem lokalen Netzwerk. USB nimmt ein neues oder nicht erreichbares Board in Betrieb. Danach wird das Board registriert und mit deinem Account verbunden.

    Wie möchtest du das Board verbinden?

    Wähle genau einen Provisioning-Weg. Du kannst den Weg später wechseln; dabei werden vorhandene Suchergebnisse verworfen.

    Geführtes Provisioning

    Board suchen

    Wichtig: WLAN funktioniert nur mit bereits provisionierten Boards. Das Board muss die GerNetiX-Basissoftware bereits besitzen und im gleichen lokalen Netzwerk erreichbar sein. Für ein neues oder nicht erreichbares Board bitte USB wählen.
    USB-Provisioning Verbinde das Board direkt mit diesem Rechner und wähle bei Bedarf den erkannten USB-Port.

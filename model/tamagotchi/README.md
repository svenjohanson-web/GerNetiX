# Tamagotchi-Modell

Dieses Verzeichnis enthaelt das fachliche Modell des Tamagotchis.

Das Modell ist die Quelle der Wahrheit. Es beschreibt Verhalten, Zustaende, Beduerfnisse und Regeln, aber keine konkrete Runtime-Implementierung.

Runtime-spezifischer Code wird daraus generiert und unter `generated/<runtime>/<projekt-id>/` abgelegt.
